# Accessible Design Co.

# Project Description
This project is a responsive website. The goal of the projectwas to build a websitethat demonstrates accessible designpractices. The site includes a hero section, navigation menu, services section, and a contact form.

The project uses semantic HTML, CSS Grid to create a responsive and accessible layout.

# How to view the project
1. Clone the repository.
2. open the project folder.
3. open "index.html" in a web browser.
